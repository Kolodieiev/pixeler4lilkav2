Даний репозиторій містить портовану прошивку з проєкту [Ідеальна консоль](https://github.com/Kolodieiev/perfect_console) для плати [Lilka_v2](https://github.com/lilka-dev/lilka).

Версія Pixeler може бути застарілою в цій прошивці. Актуальну версію фреймворку можна взяти з репозиторію "Ідеальна консоль".

Через відмінності характеристик компонентів GUI може бути портований з певними помилками відображення, а деякі можливості прошивки не мають ефекту. 

Прошивка зібрана на базі фреймворку Pixeler, документацію до якого можна знайти в каталозі `src/pixeler/doc`.

Для тестування прошивки рекомендується скопіювати вміст архіву `lilka_sd.7z` в корінь SD-карти.

Прошивка очікує, що плейлисти з музикою знаходяться в каталозі `/music/`, а серії txt-книг в каталозі `/books/`. 
Файли будь-яких налаштувань будуть зберігатися в каталозі `/.data/`.

Для керування та переміщенням по інтерфейсу використовуйте кнопки джойстика та кнопки `A`, `B`.

Коротке натискання кнопки `A` - вибір, підтвердження.
Довге утримування кнопки `A` - контекстне меню, якщо доступне, або збереження операції, наприклад, у вікну вводу.

Коротке натискання кнопки `B` - назад, відміна.
Довге утримування кнопки `B` - вихід з діалогового вікна без збереження, вихід з гри/деяких застосунків.

У грі "Комірник" коротке натискання `A` дозволяє перемикатися між вільною камерою та персонажем. Довге утримання `A` скидає рівень. Утримування `B` закриває рівень.

Для зміни розкладки клавіатури, якщо підтримується діалоговим вікном, утримуйте кнопку `SW1`(Вгору).

Встановити шпалери можна у файловому менеджері через контексте меню bmp-файла, що має коректний формат.

Запустити lua-скрипт можна у файловому менеджері через контексте меню lua-файла.

> [!WARNING]
> Прошивка збирається плагіном `Pioarduino` та не може бути зібрана за допомогою `Platformio`!

Зібраний файл прошивки та інструкція до завантаження знаходяться в каталозі `bin`.
